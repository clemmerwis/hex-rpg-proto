  <name>Task 2: Wire CharacterRenderer into Renderer and Game.js</name>
  <files>js/Game.js, js/Renderer.js</files>
  <action>
**In Game.js:**
1. Import CharacterRenderer
2. Create instance:
   ```js
   this.characterRenderer = new CharacterRenderer(this.hexGrid, GAME_CONSTANTS.ZOOM_LEVEL);
   ```
3. Set dependencies:
   ```js
   this.characterRenderer.setDependencies({
       game: this.state,
       gameStateManager: this.gameStateManager,
       inputHandler: this.inputHandler,
       hexGridRenderer: this.hexGridRenderer
   });
   ```
4. Pass characterRenderer to Renderer via setDependencies

**In Renderer.js:**
1. Remove all extracted methods (drawCharacters, drawCharacter, drawNameplate, isCharacterHovered, getFactionData)
2. Remove imports no longer needed (GAME_CONSTANTS, FACTIONS, getAnimationConfig if only used by extracted methods)
3. Store characterRenderer from setDependencies
4. In render(), replace `this.drawCharacters()` with:
   ```js
   this.characterRenderer.drawCharacters(this.ctx);
   ```
5. Remove the getFactionData delegation method added in 04-01 (no longer needed)

**Renderer.js should now be a thin orchestrator** with only:
- Constructor (config + canvas)
- setDependencies
- render() — clear, transform, drawBackground, drawHexGrid, drawCharacters, restore
- drawBackground() — background image or placeholder

That's ~75-100 lines total.
